数据解析

就目前来说，前台发送数据请求主要是from、Ajax、jsonp三种主流，当然还有其他请求方式，这里不作扩展，但是不管怎样，前台向后台请求数据都是通过http协议，所以只要数据拿到了，对于后台来说都没什么区别，可能细节上有略微差别，就像你点外卖，不管外卖小哥是骑摩托车还是电动车，你收到吃的没问题就可以了。
